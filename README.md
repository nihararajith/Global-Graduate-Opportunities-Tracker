# Global Graduate Opportunities Tracker

## Overview

Global Graduate Opportunities Tracker is a data analytics project designed to help students and fresh graduates identify international graduate opportunities, analyze hiring trends, understand skill demands, and explore visa sponsorship availability across different countries.

## Objectives

- Analyze graduate job opportunities across countries
- Identify in-demand technical skills
- Track visa sponsorship availability
- Build interactive dashboards for insights
- Support students in planning career paths

## Analysis Performed

### Country Analysis

* Analyzed the distribution of graduate opportunities across different countries.
* Identified countries with the highest number of opportunities.

### Skills Analysis

* Extracted skills from job postings.
* Identified the most frequently requested technical skills.

### SQL-Based Analysis

* Queried the MySQL database using SQL.
* Performed filtering, grouping, aggregation, and ranking operations.

## Key SQL Concepts Used

* SELECT
* WHERE
* COUNT()
* GROUP BY
* ORDER BY
* LIKE

## Current Findings

* Python is the most frequently requested skill in the sample dataset.
* SQL is the second most requested skill.
* Germany contains the highest number of opportunities in the current dataset.
* Java, Git, and Power BI appear consistently across graduate opportunities.
