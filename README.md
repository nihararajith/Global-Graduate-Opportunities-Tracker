# Global Graduate Opportunities Tracker

## Overview

Global Graduate Opportunities Tracker is a data analytics project designed to help students and fresh graduates identify international graduate opportunities, analyze hiring trends, understand skill demands, and explore visa sponsorship availability across different countries.

## Objectives

- Analyze graduate job opportunities across countries
- Identify in-demand technical skills
- Track visa sponsorship availability
- Build interactive dashboards for insights
- Support students in planning career paths

## Tech Stack

- Python
- SQL
- Power BI
- Pandas
- Git & GitHub
